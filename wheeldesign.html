An Advanced Model of a Sports Car Wheel Designed in Solidworks

Objective

After researching sports wheels in the industry, I aimed to create a complex concave wheel design using SolidWorks, leveraging its advanced features and techniques.

Project Description

This project involved creating a sophisticated wheel design and its components in SolidWorks. The design was enhanced with decals, textures, and colors.The bolts were created with chamfered ends and smoothed tops, while a helix spiral function was used to cut a sweep for ridges along with a triangle at the base. A motion study was conducted to showcase the assembly and disassembly of the wheel components.

Software Used

This project was developed using Solidworks.

Skills Learned

Through this project, I honed my skills in using SolidWorks' advanced features. These include creating intricate sketches for the revolve function, using reference planes for angled surfaces, utilizing circular patterns for mirroring functions, threading holes for bolts, and adding text with an etched look. I also learned to enhance design aesthetics with decals, textures, and colors, and to conduct a motion study for assembly and disassembly of components, shown in the gif below.
